# Pliki do GitHub Pages

## Co wrzucić do repozytorium
- `index.html`
- `style.css`
- `script.js`

## Jak uruchomić
1. Dodaj te pliki do repozytorium GitHub.
2. Wejdź w **Settings → Pages**.
3. Ustaw:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
4. Otwórz link do strony po publikacji.

## Co możesz łatwo zmienić
- teksty w `index.html`
- kolory i wygląd w `style.css`
- działanie przycisku w `script.js`
